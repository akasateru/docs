# 1. 面接対策

[面接.md](面接.md) で聞かれる質問の傾向を踏まえ、実際にどう準備するかをまとめる。

## 1.1. 事前準備

- **自己分析**: 経歴を時系列で棚卸しし、各プロジェクトでの役割・課題・成果を1〜2文で言語化しておく。
- **企業研究**: プロダクト、技術スタック、開発体制（アジャイル/ウォーターフォール、チーム規模）を事前に調べ、質問リストを作る。
- **職務経歴書との整合**: 面接で話す内容と職務経歴書に矛盾がないか確認する。

## 1.2. STARメソッドで回答を組み立てる

エピソードを聞かれたら以下の順で構造化して話す。

| 要素               | 内容                     |
| ------------------ | ------------------------ |
| Situation（状況）  | どんな状況・課題だったか |
| Task（課題）       | 自分の役割・目標         |
| Action（行動）     | 具体的に何をしたか       |
| Result（結果）     | 成果・学び               |

## 1.3. 逆質問の準備

- 技術選定の裁量、オンボーディング体制、評価制度など、事前に3〜5個は用意しておく。
- 「特にありません」は避ける。志望度の低さと受け取られやすい。

## 1.4. 模擬面接でのチェックポイント

- 結論から話せているか（PREP法）。
- 専門用語を使いすぎて相手に伝わっていないことがないか。
- 沈黙を怖がらず、考える時間を取れているか。

## 1.5. 当日の立ち回り

- カメラ映り・照明・通信環境を事前確認（オンライン面接の場合）。
- 想定外の質問には、無理に取り繕わず「少し考えさせてください」と一呼吸置く。
