## 1. 概要

- PostgreSQLは、オープンソースのリレーショナルデータベース管理システム（RDBMS）。
- 商用DBに劣らない機能と拡張性を持ち、多くのWebアプリケーションやデータ分析基盤で広く使われている。

## 2. 特徴と強み

### 2.1. 特徴

- オープンソース
  - 無償・商用利用OK・ソースコード改変可
- 標準SQL準拠
  - ANSI SQLに準拠したクエリ構文で高い互換性
- 拡張性が高い
  - ユーザー定義関数・型・演算子・言語（PL/pgSQLなど）を自由に追加可能
- ACIDトランザクション対応
  - 高い信頼性を確保（Atomicity, Consistency, Isolation, Durability）
- JSONや全文検索に対応
  - jsonb, pg_trgm, tsvector などNoSQL的機能も充実

### 2.2. 強み

- 標準SQL対応が厳密：Oracle互換を求めるシステムにも向く
- CTE、ウィンドウ関数、jsonb、配列、ENUM など高機能
- 拡張性が高い：演算子・関数・集約関数・型を自作可能
- 複雑なクエリでもパフォーマンスが出やすい（EXPLAINの出力も豊富）
- 商用DBに迫る機能がOSSで使える（例：トリガー、全文検索、論理レプリケーション）

## 3. psql

`psql -h [ホスト名] -p [ポート番号] -U [ユーザー名] -d [データベース名]`

| コマンド              | 説明                                         |
| --------------------- | -------------------------------------------- |
| **`\l`**              | データベースの一覧を表示                     |
| **`\c [DB名]`**       | 指定したデータベースに切り替え（Connect）    |
| **`\dt`**             | テーブル一覧を表示                           |
| **`\d [テーブル名]`** | テーブルの定義（カラム、型、制約）を表示     |
| **`\du`**             | ロール（ユーザー）の一覧を表示               |
| **`\df`**             | 関数の一覧を表示                             |
| **`\q`**              | psql を終了する（Quit）                      |
| **`\?`**              | 全てのバックスラッシュコマンドのヘルプを表示 |
