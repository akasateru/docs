# Mermaid

## 1. 概要

- いろいろな図が書けるツール。
- テキストで図が書ける。
- Markdownで利用可能。

以下の種類の図をサポート。

- Flowchart
- Sequence diagram
- Gantt chart
- Class diagram
- State diagram
- Pie chart
- Git graph (Experimental)
- User Journey diagram

矢印の向き
- TB: 上から下
- TD: 上から下
- BT: 下から上
- RL: 右から左
- LR: 左から右
