# 1. Cookie（クッキー）

Cookieは、Webサーバがブラウザに保存させる「サイトごとの小さなデータ」で、ステートレスなHTTPにおいてログイン状態や設定などの「状態」を保持するために使われる。関連: [HTTP](./HTTP.md)

## 1.1. Cookieの役割

HTTPは各リクエストが独立したステートレスなプロトコルなので、サーバ単体では「前にログインしたユーザか」を判別できない。これをブラウザ側の保存領域で補完するのがCookie。

- **セッション管理**: ログイン状態、ショッピングカート、ゲーム進行など
- **パーソナライズ**: 言語・地域・ダークモードなどのサイト個別設定
- **トラッキング**: 広告・アクセス解析でのユーザー行動追跡（サードパーティCookie）

## 1.2. 保存されている中身

Cookieには、ログイン中の各サイトの**セッショントークンや認証情報**が含まれる。これがあることで「ログインし直さずに」サイトを使い続けられる。

- 実体はドメインごとの `名前=値` のペア + 有効期限などの属性
- 値の多くはランダムなトークン（サーバ側のセッションと紐づく鍵）で、それ自体は人間可読な情報ではないことが多い

### 1.2.1. ディスク上は暗号化されている（macOS / Chrome）

- macOSのChromeでは、Cookieは**暗号化されて保存**されており、復号鍵はOSのキーチェーンにある。
- したがって保存ファイルを直接読んでも中身は分からず、読むには復号処理が必要。
- Cookieストアを抽出・復号して一覧化する行為は、**情報窃取型マルウェア（インフォスティーラー）**が行う手口と技術的に同じであり、自分のマシン・自分のデータであっても安易にやるべきではない。

### 1.2.2. 自分のCookieを安全に確認する方法

自分の目で特定サイトのCookieを見るだけなら、ブラウザ標準機能で十分。

- **DevTools**: 対象サイトを開く → `F12`（検証）→ `Application` タブ → `Storage` → `Cookies` → サイトURLを選択。名前・値・有効期限などが一覧表示される。
- **設定画面**: アドレスバーに `chrome://settings/cookies` → 「すべての Cookie とサイトデータを表示」。サイトごとに検索・個別削除も可能。
- **鍵アイコン**: アドレスバー左の鍵アイコンから、今開いているサイトのCookieだけを手早く確認できる。

## 2. Cookieをクリアするとどうなるか

Cookieを削除すると、ブラウザとサイトの間の「接続状態」がリセットされる。影響の大きい順に整理する。

## 2.1. 起きること

- **ログインが軒並み切れる**（最も体感する変化）
  - 多くのサイトはログイン状態をセッションCookieで保持しているため、削除すると各サイトからログアウト状態になる。次回アクセス時に再度ログイン（＋2段階認証）を求められる。
- **サイトの設定・状態がリセットされる**
  - ゲスト状態のカート、言語・ダークモードなどの個別設定
  - 「Cookie同意バナーを閉じた記録」→ 再表示される
  - 「このデバイスを信頼する」の記録 → 2段階認証を再度求められる
- **パーソナライズが一旦リセットされる**
  - 追跡Cookieも消えるため、プライバシー的にはプラス（トラッキングがリセットされる）。

## 2.2. クリアしても消えないもの（誤解しやすい点）

- **保存済みのパスワード**（ブラウザのパスワードマネージャー内）は消えない → だから再ログインは簡単。
- **ブックマーク・履歴・ダウンロードファイル**は別物なので残る。
- **アカウント自体**が消えるわけではない。あくまでブラウザとサイト間の状態が切れるだけ。

## 2.3. 関連項目との区別

Chromeの「閲覧履歴データの削除」画面には似た項目が並ぶので区別しておく。

| 項目 | 消すとどうなる |
| --- | --- |
| Cookie とサイトデータ | ログアウト等（上記のとおり） |
| キャッシュされた画像とファイル | サイトの読み込みが一時的に遅くなるだけ。ログインは切れない |
| 閲覧履歴 | 訪問記録が消える。ログインには影響なし |
| 保存したパスワード | ログイン情報そのものが消える（通常はチェックしない） |

## 2.4. 使いどころ

- **サイトの不具合時**（表示崩れ・ログインループ）→ そのサイトのCookieだけ消すと直ることが多い。全消しより影響が小さい。
- **共用・他人のPCを使った後** → ログアウト状態にできる。
- **プライバシーをリセットしたいとき** → 追跡Cookieを消せる。
