# 1. Wi-Fi通信とデータ通信の基礎

「光回線の1ギガ/10ギガ」と「スマホプランの30GB」のように混同しやすい通信の単位（速度と容量）を整理し、Wi-Fi規格や用途別の目安をまとめる。

## 1.1. 「1ギガ」には2つの意味がある

回線プランやスマホプランで使われる「ギガ」は、文脈によって**まったく別の指標**を指す。

| 表現 | 正式な単位 | 意味 | 例 |
| --- | --- | --- | --- |
| 光回線の「1ギガ / 10ギガ」 | **Gbps**（ギガビット毎秒） | **通信速度**（1秒間に流せるデータ量） | ドコモ光1ギガ = 最大1Gbps |
| スマホプランの「30GB」 | **GB**（ギガバイト） | **月間データ容量**（月に使える総量） | ahamo = 30GB/月 |

- **bit（ビット）と Byte（バイト）**: `1 Byte = 8 bit`。速度は bps（bit毎秒）、容量は Byte で表すのが慣例。
  - 例: 1Gbps の回線は理論上 **1秒あたり125MB**（1,000Mbit ÷ 8）転送できる。
- 光回線は**使い放題**（容量無制限）なので速度だけがプランの差。スマホ回線は容量が上限で、超えると速度制限がかかる。

## 1.2. 通信速度（bps）の目安

### 1.2.1. 用途別に必要な速度

| 用途 | 必要な速度の目安 |
| --- | --- |
| Web閲覧・SNS | 1〜10Mbps |
| ビデオ会議（Zoom/Meet） | 3〜10Mbps |
| フルHD動画（YouTube/Netflix） | 5〜10Mbps |
| 4K動画ストリーミング | 25Mbps前後 |
| オンラインゲーム | 速度より**低遅延（Ping値）**が重要。30Mbps程度で十分 |
| ゲームや大容量ファイルのダウンロード | 速いほど短時間で完了（ここだけ速度が効く） |

- **1Gbps（実効でも数百Mbps）あれば家庭用途はほぼすべて余裕**。10ギガ回線が活きるのは、大容量ダウンロードを頻繁にする・同時接続端末が非常に多い場合くらい。
- **ベストエフォート**: 「最大1Gbps」は理論値。実効速度は混雑や機器性能で下がり、実測100〜500Mbps程度出ていれば良好。
- 速度計測は [fast.com](https://fast.com/ja/) や [Speedtest](https://www.speedtest.net/) で確認できる。

### 1.2.2. ボトルネックは回線よりルーター・接続方式のことが多い

- **Wi-Fiルーターの規格が古いと回線速度を活かせない**（下記 1.4 参照）。
- **IPoE（IPv4 over IPv6）対応**か: 従来のPPPoE方式は網終端装置で混雑しやすく、夜間に速度が落ちる。IPoE対応ルーター・プロバイダなら混雑を回避できる。
- 有線LANケーブルにも規格がある: 1Gbps以上を使うなら **Cat5e以上**（Cat6/6A推奨）。

## 1.3. データ容量（GB）の目安

月間データ容量「1GB / 10GB / 30GB」でできることの感覚値。

| 用途 | 1GBでできる量の目安 |
| --- | --- |
| Web閲覧 | 約3,000〜5,000ページ |
| LINE・メール（テキスト中心） | 実質無視できる消費量 |
| 音楽ストリーミング（標準音質） | 約20時間 |
| YouTube（480p） | 約2時間 |
| YouTube（1080p/フルHD） | 約40分〜1時間 |
| ビデオ会議 | 約1.5〜2時間 |
| 地図アプリ | 約1,300回の経路検索 |

- **動画とビデオ会議が圧倒的に容量を食う**。テキスト中心なら月1〜3GBで足りるが、外で動画を見る習慣があると月20GB以上いくことも。
- **自宅にWi-Fi（光回線）があればスマホの容量消費は激減**する。動画・アプリ更新・クラウドバックアップを自宅Wi-Fiに寄せるのが基本戦略。
- 自分の実際の使用量は My docomo などキャリアのマイページで月別に確認できる。プラン選びは「直近3ヶ月の実績 + 余裕2〜3割」が目安。

## 1.4. Wi-Fi規格の整理

| 規格（世代名） | 規格名 | 最大速度（理論値） | 周波数帯 |
| --- | --- | --- | --- |
| Wi-Fi 4 | IEEE 802.11n | 600Mbps | 2.4GHz / 5GHz |
| Wi-Fi 5 | IEEE 802.11ac | 6.9Gbps | 5GHzのみ |
| **Wi-Fi 6** | IEEE 802.11ax | 9.6Gbps | 2.4GHz / 5GHz |
| Wi-Fi 6E | 802.11ax拡張 | 9.6Gbps | + 6GHz |
| Wi-Fi 7 | IEEE 802.11be | 46Gbps | 2.4 / 5 / 6GHz |

- **1Gbps回線には Wi-Fi 6 対応ルーターで十分**。Wi-Fi 6 は多台数同時接続に強く（OFDMA）、省電力（TWT）。
- **周波数帯の使い分け**:
  - **2.4GHz**: 遠くまで届き障害物に強いが、遅くて電子レンジ等と干渉する。
  - **5GHz**: 速くて干渉が少ないが、壁に弱く距離で減衰する。ルーターの近くでは5GHz推奨。
- ルーターの買い替え目安は4〜5年。セキュリティは **WPA3**（なければWPA2）を使う。

## 1.5. プロバイダ（ISP）とは

**プロバイダ（ISP: Internet Service Provider）は、回線をインターネット網に接続する役割を担う事業者**。インターネット接続は「回線」と「接続」の2つの役割に分かれている。

| 役割 | やること | 例 |
| --- | --- | --- |
| **回線事業者** | 自宅までの物理的な光ファイバー網を提供（道路にあたる） | NTT東日本/西日本（フレッツ光）、KDDI、電力系 |
| **プロバイダ（ISP）** | 回線をインターネット網に接続（高速道路への入口にあたる） | OCN、BIGLOBE、So-net、GMOとくとくBB など |

- 光ファイバーが家まで来ているだけではインターネットに出られない。プロバイダが**インターネットへの接続口・IPアドレスの割り当て・メールアドレス提供**などを担って初めて通信できる。

### 1.5.1. 契約形態の変遷

- **昔（フレッツ光時代）**: 回線は NTT、プロバイダは別途契約と、**2契約・2請求**が必要だった。
- **今（光コラボレーション）**: ドコモ光や ahamo光のように、事業者が NTT 回線を借りてプロバイダとセットで提供する形が主流。**契約も請求も1本**になる。
  - ドコモ光: 契約時にプロバイダ（OCN、GMO など）を選ぶ方式
  - ahamo光: **プロバイダ一体型**（OCNインターネット相当）で選ぶ必要すらない方式

### 1.5.2. 接続方式（PPPoE / IPoE）はプロバイダ側の話

- 1.2.2 で触れた **IPoE（IPv4 over IPv6）** は、プロバイダの接続方式のこと。
- **PPPoE（従来方式）**: プロバイダの網終端装置を経由するため、夜間など利用集中時に混雑して速度が落ちやすい。
- **IPoE**: 網終端装置を経由せず混雑を回避できるため速度が安定する。ahamo光は IPoE 標準対応。
- ルーター選びで「IPoE（IPv4 over IPv6）/ OCNバーチャルコネクト対応」が条件になるのは、この方式にルーター側も対応している必要があるため。

## 1.6. まとめ: プラン選びの考え方

- **光回線（速度で選ぶ）**: 家庭用途なら1ギガで十分。10ギガは価格差に見合う用途があるかで判断する。
- **スマホ（容量で選ぶ）**: 自宅Wi-Fiがあるなら実績ベースで小〜中容量（〜30GB）に寄せられる。
- 実際の見直し記録は `private-docs/life/固定費見直し.md`（ドコモ光10ギガ → ahamo光1ギガ移行）を参照。
