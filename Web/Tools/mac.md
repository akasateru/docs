# 2. バッテリーの充電管理と劣化対策

MacBookを充電器に挿しっぱなし（据え置き運用）にする場合、劣化を抑えられるかどうかは「バッテリー健康管理」の設定次第。

## 2.1. バッテリー健康管理の設定項目

システム設定 → バッテリー → バッテリーの状態（ⓘ）から選べる。

- **標準**: 常に100%まで充電し続ける。挿しっぱなしだと劣化要因になりやすい。
- **最適化された充電**: 普段の利用パターンを学習し、使う直前まで80%程度で待機してから100%まで上げる。
- **80%の制限**: 常に80%までしか充電しない。据え置き・挿しっぱなし運用なら劣化抑制効果が一番大きい。

据え置きでほぼ常時給電するなら「80%の制限」が最も劣化を抑えられる。

## 2.2. ターミナルでバッテリー状態を確認する方法

GUIの設定項目名はmacOSバージョンによって変わる（例: macOS 26 Tahoeでは「最適化された充電」の表示場所・文言が旧バージョンと異なる場合がある）ため、実際に効いているかはコマンドで裏取りできる。

- `pmset -g batt`: 現在の充電状態（充電中か、残量、AC/バッテリー駆動か）
- `system_profiler SPPowerDataType`: バッテリーのサイクルカウント・健康状態(Condition)・最大容量(Maximum Capacity)
- `ioreg -rc AppleSmartBattery -w0`: 詳細な生データ。特に `DailyMaxSoc` / `DailyMinSoc` は直近24時間の充電率の上限・下限を示し、これが常に99〜100%付近なら「80%の制限」や「最適化された充電」が実質効いていない（＝設定がOFF、または反映されていない）ことの裏付けになる

## 2.3. 現状のログ（2026-07-13時点）

- 対象機: MacBook Air (Mac17,3 / Apple M5) macOS 26.5.2
- サイクルカウント22、健康状態 Normal/Good、最大容量100%（劣化なし、新品に近い）
- `DailyMaxSoc`/`DailyMinSoc` が 99〜100% で推移 → 充電制限系の機能は実質効いていない状態だった
- 対策として「80%の制限」を設定でONにすることを推奨した
- バッテリー健康ログに一度だけ "Check Battery" への遷移が記録されていたが、直後に "Good" へ復帰。単発のセンサー誤検知の可能性が高く、再発しなければ様子見でよい
