## 2. 仕組み

- PCやスマホの通信はすべて、世界中に点在するZscalerのデータセンター（ZIA/ZPA）を経由するようになる
  - ZIA (Zscaler Internet Access): インターネットやSaaS（Microsoft 365, Salesforceなど）への通信を検査する。
  - ZPA (Zscaler Private Access): 社内システムやクラウド（AWS/Azure）上の個人用サーバーへの通信を制御する。

- 通信が発生するたびに、以下のチェックを瞬時に行う。
  - 本人確認: OktaなどのID管理ツールと連携し、多要素認証を行う。
  - 端末確認: 会社支給のPCか、OSは最新か、ウイルス対策ソフトは動いているかをチェック。
  - 内容検査: 通信の中にウイルスが隠れていないか、機密情報が漏洩（DLP）していないかをリアルタイムでスキャン。

- Zscalerの最大の特徴の一つは、**「内側から外側への通信（アウトバウンド）」**しか許可しない点。
