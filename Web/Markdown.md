# Markdown

## 1. 概要

Markdownは、文章をいい感じにデコレーションできるエンジニア愛用のツール。

## 2. 構文

これらさえ覚えておけば、ほとんどのドキュメントを作成できる。

| 機能         | 記述方法                     | 結果                |
| ------------ | ---------------------------- | ------------------- |
| **見出し**   | `#` タイトル1 `##` タイトル2 | ～相当              |
| **強調**     | `**強調**`                   | **強調**            |
| **斜体**     | `*斜体*`                     | _斜体_              |
| **リスト**   | `-` 項目A `-` 項目B          | ・項目A ・項目B     |
| **番号付き** | `1.` 項目1 `2.` 項目2        | 1. 項目1 2. 項目2   |
| **引用**     | `> 引用文`                   | 引用文              |
| **リンク**   | `[文字](URL)`                | 文字                |
| **画像**     | `![代替文字](画像URL)`       | 画像の埋め込み      |
| **コード**   | `` `code` ``                 | `code` (背景色付き) |
| **水平線**   | `---`                        | 水平線              |
